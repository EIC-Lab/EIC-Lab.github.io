# EIC Lab Website

Static website for EIC Lab, designed for GitHub Pages deployment.

## Deploy to GitHub Pages

1. Create a GitHub organization or user account. GitHub names cannot contain underscores, so use a name such as `eic-lab` or `EICLab`.
2. Create a repository named `<account>.github.io`, for example `eic-lab.github.io`.
3. Push this repository to GitHub:

```bash
git remote add origin git@github.com:<account>/<account>.github.io.git
git push -u origin main
```

4. Open `https://<account>.github.io`.
